description

this is cursed serenity, a university project showcased in the UTS 2025 tech fest made for the game design methodologies class in unity! this game is a platformer where you play as a shapeshifting princess tasked with saving her animal friends from evil scientists.

this was my first time creating a game in a group which was both challenging and rewarding. i learnt a lot about collaborative programming, and the importance of leaving comments and "templates" in my code to help other programmers build off of my work. this is also where i got a lot of my practice bugfixing and learnt about the importance of playtesting. sometimes i would not notice a bug until i watched a friend play the game in the most absurd and unpredictable way!
