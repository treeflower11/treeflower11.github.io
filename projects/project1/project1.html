description

this is derpy eco sim, a university project made for the advanced games programming class in unreal engine! derpy eco sim is a tech demo where you can observe different kinds of animals (carnivorous, herbivorous, or omnivorous) interacting with one another.

in this project i learnt how to program an AI behaviour tree to control the behaviour of animals based on different factors such as hunger and health values. i also learnt about multiplayer concepts such as replication and applied this to the project to ensure that every player had an accurate representation of the game, which was particularly tricky with the UI graph that i created for this project. this was also my first time using unreal engine where i used a combination of c++ and blueprints to create this tech demo!

another component of my university assignment included creating videos to explain my work. i have attached a link to my video explaining how i created the AI behaviour tree on this page! i had a lot of fun trying to break down my code into easier-to-digest analogies and using silly graphics to explain my thought process.
