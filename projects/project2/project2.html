description

this is trash game, a short game jam project that i made with one of my friends in unity! trash game is a fighting game where you summon weapons (or useless trash) from a mystical slot machine to fight your friend to the death.

overall this game was quite silly and experimental, purely made to relax during the university semester. we aimed to create a game that we could envision seeing on a flash player site such as coolmathsgames. some of the technical things that i personally learnt while creating this game was the usefulness of inheritance when programming a system and how beneficial it is to other programmers when working collaboratively. it saved a lot of redundant code, and was helpful in structuring this system where we needed to randomly select objects from a list.
